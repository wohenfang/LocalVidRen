# LocalVidRen - 本地短视频智能重命名系统

## 项目简介

LocalVidRen 是一个 Windows 专属的本地短视频桌面应用程序，通过 AI 模型分析 + 语音识别技术，快速生成短视频内容总结，并自动将总结作为视频的新文件名。

### 核心特性
- **100% 本地隐私**：数据完全不出电脑
- **秒级分析**：针对 500M 以内短视频优化
- **自动化处理**：文件夹监控 + 无人值守运行
- **灵活模型选择**：支持本地模型和在线 API

## 系统要求
- 操作系统：Windows 10/11 (64 位)
- Python 3.11+
- 内存：4GB+

## 功能列表
- 视频文件扫描与管理
- AI 模型内容分析（本地/API 双模式）
- 自动重命名
- 批量处理与自动化
- 文件夹实时监控
- 无人值守运行
